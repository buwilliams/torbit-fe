Torbit FE Challenge Server
===

Prerequisites
---

* Node.js

Getting Started
---

* `git clone https://github.com/buwilliams/torbit-fe.git`
* `cd torbit-fe`
* `npm install`
* `bower install`
* `gulp serve` (or for production: `gulp build && gulp serve:prod`)
* Open browser [http://localhost:9000](http://localhost:9000)
